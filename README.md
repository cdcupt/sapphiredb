## SapphireDB
[![Build Status](https://travis-ci.org/cdcupt/sapphiredb.svg?branch=master)](https://travis-ci.org/cdcupt/sapphiredb)
A distributed metafile system
----------
## Getting Started
### Download
```
git clone https://github.com/cdcupt/sapphiredb.git
```
### Install
```
mkdir build
cd build
cmake ..
make
```
